# Para onde sua vida vai?

Aplicação de reflexão pessoal que compara a direção dominante da vida, o retorno desejado, o retorno percebido e os possíveis custos ocultos. Criada com Next.js, TypeScript e Tailwind CSS.

## Rodar localmente

```bash
npm install
npm run dev
```

## Validar para produção

```bash
npm test
npm run lint
npm run build
```

## Publicar na Vercel

Importe este diretório como um novo projeto na Vercel. O framework e os comandos são detectados automaticamente e nenhuma variável de ambiente é necessária.

As respostas e o progresso permanecem somente no navegador da pessoa, usando `localStorage`. A resposta aberta não participa da interpretação automática nem da imagem compartilhável.
