# No que sua vida está transformando você?

Aplicação de reflexão pessoal criada com Next.js, TypeScript e Tailwind CSS.

## Rodar localmente

```bash
npm install
npm run dev
```

## Validar para produção

```bash
npm run lint
npm run build
```

## Publicar na Vercel

Importe este diretório como um novo projeto na Vercel. O framework e os comandos são detectados automaticamente e nenhuma variável de ambiente é necessária.

As respostas e o progresso permanecem somente no navegador da pessoa, usando `localStorage`.
